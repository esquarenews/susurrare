Home

Use Home to check weekly performance, trends, and quick next actions.

  1. Open Home from the sidebar.
  2. Review key stats: speed, words, apps used, and minutes saved.
  3. Select View my stats for a weekly trend breakdown.

Modes

Modes let you tune model, insertion style, and formatting for each workflow.

  1. Select Modes and create a new mode or edit an existing one.
  2. Set model type and streaming on/off based on latency vs. accuracy needs.
  3. Choose insertion behavior and optional rewrite prompt.
  4. Enable shortcuts/punctuation/formatting toggles as needed.

Vocabulary

Vocabulary replacements correct names, acronyms, and recurring terms.

  1. Add a source phrase and replacement.
  2. Set scope to Global or Per mode.
  3. Save and test by dictating the source phrase again.

Shortcuts

Shortcuts map spoken keywords to reusable snippets.

  1. Add a spoken keyword and the snippet text to insert.
  2. Choose global or mode-specific scope.
  3. Use Edit to maintain snippets over time.

Configuration

Configuration controls app behavior, language, hotkeys, updates, and API key.

  1. Set your theme and recording window preferences.
  2. Configure timeout, language, and clipboard restore behavior.
  3. Set push-to-talk/toggle/cancel/change-mode hotkeys.
  4. Save and verify your OpenAI API key status.

Sound

Sound settings manage microphone behavior and audible recording cues.

  1. Toggle automatic gain and silence removal.
  2. Enable sound effects for start/stop confirmation.
  3. Adjust cue volume to match your environment.

Models Library

Review available transcription models and test pinned model IDs.

  1. Open Models Library to review speed/accuracy options.
  2. Use pinned model ID for advanced/custom setups.
  3. Validate model ID format before using in a mode.

History

History keeps local dictations available for reuse, pinning, and export.

  1. Search dictations and select multiple entries.
  2. Copy, re-insert, pin, or delete entries as needed.
  3. Export selected items to a local text file.

Diagnostics

Diagnostics exports troubleshooting data for support investigation.

  1. Open Diagnostics.
  2. Select Export diagnostics.
  3. Share the generated archive with support when requested.
